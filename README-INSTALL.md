# دليل تنصيب نظام الأرشفة الإلكترونية
## كلية الصيدلة — جامعة البصرة

---

## متطلبات التشغيل

| المتطلب | الإصدار | ملاحظة |
|---------|---------|--------|
| Windows | 10 / 11 | 64-bit فقط |
| Node.js | 18 أو أحدث | https://nodejs.org |
| Git | أي إصدار | https://git-scm.com |
| السكانر | TWAIN/WIA | أي سكانر يدعم Windows |

---

## خطوات التنصيب للمطور

```bash
# 1. نسخ المشروع
git clone https://github.com/your-org/pharmacy-archive.git
cd pharmacy-archive

# 2. تنصيب المكتبات
npm install

# 3. تشغيل البرنامج مباشرة
npm start
```

---

## بناء ملف التنصيب (.exe)

```bash
# بناء مثبّت NSIS كامل
npm run build

# أو بناء نسخة محمولة (لا تحتاج تنصيب)
npm run build-portable
```

بعد البناء ستجد الملفات في مجلد `dist/`:
- `أرشيف كلية الصيدلة Setup 1.0.0.exe` — مثبّت كامل
- `أرشيف كلية الصيدلة 1.0.0.exe` — نسخة محمولة

## هيكل البيانات عند التشغيل

```
C:\Users\{اسم المستخدم}\AppData\Roaming\pharmacy-archive\
    └── pharmacy_archive.db          ← قاعدة البيانات SQLite

C:\PharmacyArchive\                  ← (قابل للتغيير من الإعدادات)
    ├── شهادات_ودرجات\
    ├── قبول_وتسجيل\
    ├── وثائق_إدارية\
    ├── أبحاث_ورسائل\
    ├── عقود_واتفاقيات\
    ├── وثائق_أخرى\
    └── Backup\
```

## إعداد السكانر

1. وصّل السكانر بالكمبيوتر عبر USB
2. نصّب تعريف السكانر من القرص المرفق أو موقع الشركة
3. افتح البرنامج → قسم "المسح الضوئي"
4. اضغط "تحديث الأجهزة" — سيظهر اسم السكانر تلقائياً
5. اختر الجهاز واضغط "بدء المسح"

### السكانرات المختبرة
- Canon DR-C225 / DR-C225W
- Fujitsu ScanSnap iX1600
- HP ScanJet Pro 2000
- Epson DS-530
